# Flask project

```sh
$ cd FlaskProject
```

### Setup a virtual environment 
```sh
$ virtualenv env
$ source env/bin/activate
```

### Install dependencies
```sh
$ pip install -r requirements.txt
```

### Setup flask path
```sh
$ export FLASK_APP=flaskr/flaskr.py
```

### On development
```sh
$ export FLASK_DEBUG=true
```

### Run
```sh
$ flask run
```

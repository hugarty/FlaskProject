http://flask.pocoo.org/docs/0.12/installation/#installation

### Install requests
http://docs.python-requests.org/en/master/user/install/#install

### Run
```
/flask/flaskr$ export FLASK_APP=flaskr.py
```
```
/flask/flaskr$ export FLASK_DEBUG=true
```
```
$ flask run
```
